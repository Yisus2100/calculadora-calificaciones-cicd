# Secuencia sugerida para la demostración

1. Copiar los archivos de `versiones/v1.0.0` al proyecto de trabajo.
2. Ejecutar `npm install` y `npm run verify`.
3. Crear el repositorio, los workflows y la etiqueta `v1.0.0`.
4. Crear el Issue usando `issue.md`.
5. Crear la rama `feature/clasificacion-academica`.
6. Sustituir temporalmente las pruebas por `versiones/paso-fallo/calificaciones.test.js`.
7. Ejecutar y enviar el fallo controlado.
8. Incorporar la versión final ubicada en la raíz del proyecto.
9. Ejecutar `npm run verify` y crear el Pull Request con `pull_request.md`.
10. Integrar, observar el despliegue, etiquetar `v1.1.0` y completar la matriz.
